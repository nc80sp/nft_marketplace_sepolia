2. Mint (/mint) - Create New NFT

画像・名前・説明を入力してNFTをミントするフォーム。IPFS upload + on-chain mint。

localhost:5173/mint
NFT Marketplace
Home Mint My NFTs
0x1234...abcd 1.2345 ETH

Mint NFT

📎 Drop image here or click to upload
PNG, JPG, GIF (max 10MB)

ℹ MetaMask will ask for transaction confirmation

Processing states:
1. "画像をIPFSにアップロード中..." (uploading image)
2. "メタデータをIPFSにアップロード中..." (uploading metadata JSON)
3. "NFTをミント中..." (waiting for TX confirm)
4. "ミント成功!" (done)
Wallet not connected: button disabled, shows "ウォレットを接続してください"
Validation: Name and Image are required fields

Not listed card: price input + "List" button (approve + listItem TX)
Listed card: green "Listed" badge + price + red "Cancel Listing" button
Listing flow: List click → MetaMask approve TX → MetaMask listItem TX → refresh

Header: "Connect Wallet" button (blue) replaces address display
Home page: listings are visible without wallet (read-only)
Mint / My NFTs: shows connect prompt as above
NFT Detail: shows details but Purchase button is disabled
